Fall of Kowe - How To Play The Game

About The Game

Fall of Kowe is a thrilling space adventure in which the aim of the
game is to stay alive and get the best score.

Characters/Pickups

Kowe - Main Character

Enemies (to avoid) - Debris

Red Coin - Decrease score by 250pt

Blue Coin - Increase score by 250pt

Controls

Up Arrow Key - Move Up
Up Arrow Key - Move Down
Left Arrow Key - Move Left
Right Arrow Key - Move Right
Space Bar - Starts the Game
